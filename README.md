# CraftHarmony Next.js

Premium handicraft eCommerce website converted from React + Vite to Next.js App Router.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build for production

```bash
npm run build
npm run start
```

## Included

- Next.js App Router
- Tailwind CSS styling
- Separate pages for Privacy Policy, Refund Policy, and Terms & Conditions
- Product listing and product detail routes
- Cart with localStorage persistence
- Responsive premium UI
- Image fallback support so missing external images do not break the layout
